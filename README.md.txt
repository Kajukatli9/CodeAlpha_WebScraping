# 📚 Book Store Web Scraper using BeautifulSoup

## 📌 Project Overview
This project is developed as part of the **CodeAlpha Data Analytics Internship**.

The program scrapes book information from the website **Books to Scrape** using Python and BeautifulSoup.

## 🎯 Objective
To extract structured data from a website and save it into a CSV file for further analysis.

## 🌐 Website Used
https://books.toscrape.com/

## 🛠️ Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas
- Google Colab
- Matplotlib

## 📊 Data Collected
- Book Title
- Price
- Rating
- Availability

## 📈 Project Features
- Scrapes all **50 pages**
- Collects information of **1000 books**
- Stores data in CSV format
- Performs basic data analysis
- Visualizes rating distribution using a bar chart

## 📂 Output
- scraped_books_1000.csv
